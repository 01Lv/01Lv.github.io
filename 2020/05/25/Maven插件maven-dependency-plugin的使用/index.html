Maven插件maven-dependency-plugin的使用

发表于 2020-05-25 分类于 Maven Valine：
本文字数： 15k 阅读时长 ≈ 14 分钟

在日常开发中有可能碰到包冲突的问题！

mvn dependency:list: 列出项目最终解析到的依赖列表

mvn dependency:tree : 打印出依赖树

+- 符号表示该包后面还有其它依赖包

\- : 表示该包后面不再依赖其它 jar 包
