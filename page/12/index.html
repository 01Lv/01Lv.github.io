Java常量池理解与总结

Java 中的常量池主要分为 Class文件常量池，运行时常量池，全局字符串常量池

Class 文件常量池

class 文件是一组以字节为单位的二进制数据流，在 java 在java代码的编译期间，我们编写的java文件就被编译为.class文件格式的二进制数据存放在磁盘中，其中就包括class文件常量池。 class文件中存在常量池（非运行时常量池），其在编译阶段就已经确定。
