关于我

“世间所有的相遇，都是久别重逢”

本人是一名90后的程序员，热爱网络编程，折腾技术,总是喜欢尝鲜，只要看到大神的博客有意思的点，都会想着去落地实现，嘿嘿！擅长于 Java 后端的开发以及 Linux 服务端的折腾，其实也钟意于前端通过代码能写出许许多多出乎意料，高大上，美轮美奂的前端界面。
